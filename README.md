## Descrição 

o projeto consiste na composicao de uma tabela que conta com capitais brasileiras e a distancia entre essas capitais.as capitais constante nesse projeto sao:

* Manaus 
* Natal
* Palmas
* Porto alegre
* Porto velho
* Recife
* Rio branco
* Rio de janeiro
* Salvador
